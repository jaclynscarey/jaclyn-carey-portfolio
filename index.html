Trivia Game - Are You Smarter Than ChatGPT???

HTML5 | CSS3 | JavaScript
This is a 1 player trivia game based on the gameshow "Are you smarter than a 5th grader?". But instead, the questions and answers were generated by ChatGPT. The topics of the questions range from math, science, history, geography, literature, and 90's movies. Questions are randomly selected from a pool of 106 questions with each new game. The game's layout is responsive, which makes it possible to be played on desktop or mobile devices.
